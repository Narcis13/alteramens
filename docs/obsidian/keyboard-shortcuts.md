# Obsidian Keyboard Shortcuts

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+O` | Quick switcher (open file) |
| `Cmd+P` | Command palette |
| `Cmd+Shift+F` | Search in all files |
| `Cmd+E` | Toggle edit/preview mode |
| `Cmd+Click` | Open link in new pane |

## Editing

| Shortcut | Action |
|----------|--------|
| `Cmd+B` | Bold |
| `Cmd+I` | Italic |
| `Cmd+K` | Insert link |
| `Cmd+]` | Indent |
| `Cmd+[` | Outdent |

## Panes

| Shortcut | Action |
|----------|--------|
| `Cmd+\` | Split vertically |
| `Cmd+Shift+\` | Split horizontally |
| `Cmd+W` | Close current pane |

## Quick tips

- Use `[[` to start linking to another note
- Use `#` in links for heading anchors: `[[note#heading]]`
- Use `|` for display text: `[[note|display text]]`
