# SuperX (superx.so)

X (Twitter) growth platform care unifică AI drafting + scheduling + engagement automation + analytics. Built de indie makers Tibo + Rob, livrat ca Chrome extension + web app. Claim: "9,000+ creators, 5B+ impressions."

## Poziționare în piață

Tabără **automation-first** (alături de Tweet Hunter, Hypefury, Hookline). Opus tabărei human-in-loop reprezentată de [[typefully]] și (intern) Semnal.

## Feature stack

| Strat | Features |
|---|---|
| Drafting | AI Post Writer, AI Thread Writer, "voice-matched" rewrite |
| Discovery | Daily Viral Inspiration, 10M+ viral posts library, "Ready to Post" stream |
| Scheduling | Smart Scheduler la "ora optimă a audienței" |
| **Automation** | **Auto Retweet, Auto Plug (CTA injection), Auto DM, Auto Delete low-performers** |
| Engagement | Strategic Engagement Feed, Unified Mentions Hub |
| Analytics | Performance Breakdown, Audience Behavior, Chrome ext in-feed |

## Pricing (2026-04)

- **Pro** $39/mo — 5 conturi, scheduling, 1.000 Auto-DM/lună
- **Advanced** $39-$49/mo — AI writers, 750 posts/lună, 1.000 AI credits/zi
- **Ultra** $199/mo — 10 conturi, 2.000 posts/lună, 2.500 Auto-DM/lună
