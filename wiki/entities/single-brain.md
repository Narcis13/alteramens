# Single Brain

Unified vector database built by [[single-grain]] that ingests all company data every 15 minutes. The practical implementation of the [[world-model]] concept from Dorsey's framework.

## What It Ingests

Slack messages, CRM records, Gong call transcripts, Google Analytics, Search Console data, client deliverables, meeting notes, financial data. Every agent in the system queries this same brain.

## Why It Matters

The data structure that lets AI understand your specific business. Not the model itself — the context layer. Takes months to build because data has to accumulate. This accumulation IS the moat ([[data-compounding-moat]]).

## Connection to Faber

Faber wiki is Alteramens' version of a world model — structured, compounding, cross-referenced knowledge that agents can query. Different implementation (markdown wiki vs vector DB), same principle: proprietary context accumulation as competitive advantage.

## Product Evolution

Started as internal tool → became a product sold to clients at singlebrain.com. Classic [[internal-to-product]] pattern.
