03The match · how a request moves

A request enters, five ideas leave.

Tickets — each one an operator's question — travel through five stations. Most carry forward; some are kicked back to ASK when context is too thin to filter against. What lands at OUTPUT is three to five tailored ideas, not the whole library.

07Output etiquette · the shape of an answer

Five facets per recommendation.

The strategist's answer has a shape. Each recommended idea carries five facets — name, fit, first steps, expected outcome, and resources. Nothing less; rarely more.

A

Idea name · one-line description.

The canonical name and a single sentence the operator can repeat verbatim to a colleague.

B

Why it fits · the connection.

Tie it back to the operator's stated stage, audience, or constraint. If you can't, the idea didn't earn the slot.

C

How to start · first two-or-three steps.

Concrete. Not "build a strategy" — the literal next action they should do this week.

D

Expected outcome · what success looks like.

Name the leading indicator: signups per week, share of voice, qualified inbound. Hand-wavy outcomes get hand-wavy execution.

E

Resources needed · time, budget, skills.

Honest count of hours, dollars, and people. The operator decides whether the price is right; the strategist doesn't hide it.

F

Defer to the channel skill.

For execution, hand off: programmatic-seo, competitor-alternatives, email-sequence, free-tool-strategy, referral-program. This skill names the move; the channel skill makes it.
