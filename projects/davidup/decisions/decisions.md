# DavidUp - Decizii

| Decizie | Rațiune | Outcome |
|---------|---------|---------|
| JSON-to-video approach | Programmatic, API-first, AI-friendly | Good |
| Canvas API for rendering | Browser-native, no heavy deps | Good |
| FFmpeg for encoding | Industry standard, reliable | Good |
| Hono.js framework | Lightweight, Bun-compatible, great DX | Good |
| Zod for all schemas | Type-safe validation, shared frontend/backend | Good |
| API Key auth (not JWT) | Simpler for API service, stateless | Good |
| Platform presets | Reduces user decisions, better defaults | Good |
| Keyframe animation system | Flexible, covers most motion needs | Good |
| Claude for template generation | Natural language → video spec | Good |
