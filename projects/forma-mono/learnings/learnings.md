# Forma - Ce am învățat

## Technical

- Bun workspaces work well for monorepo cu packages interdependente
- Zero-dependency constraint forces better design decisions
- ToolSchemaGenerator este killer feature - bridge between forms and LLM tools
- Recursive validation pentru group/array fields adaugă complexitate dar e esențial
