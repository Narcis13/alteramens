# CCT2 - Decizii

| Decizie | Rațiune |
|---------|---------|
| Next.js (not plain React) | Server-side rendering needed for video generation |
| Remotion for video | React-based video creation, programmatic |
| App Router (not Pages) | Modern Next.js patterns |
| Server-side rendering | Heavy video encoding shouldn't happen client-side |
| Template registry pattern | Extensibil, ușor de adăugat template-uri noi |
| Dark theme + purple | Fits subliminal/meditation aesthetic |
